# Better Wallet 🧠💸

**Control your crypto with natural language.**  
Better Wallet is an AI-native Solana wallet that lets you type commands like:

> "Send 2 SOL to Alice"  
> "Create a token called MyCoin"  
> "Check my balance"

...and it just works.

## ✨ Features

- Natural language wallet actions (send, receive, create token, swap, etc.)
- Built-in support for Solana transactions
- Secure intent parsing via OpenAI LLM
- Early automation support through customizable "workers" (e.g., stop-loss)
